Back-End Jekins


Jenkins es un servidor open source para la integración continua. Es una herramienta que se utiliza para compilar y probar proyectos de software de forma continua, lo que facilita a los desarrolladores integrar cambios en un proyecto y entregar nuevas versiones a los usuarios.

Es necesario comenzar con una descripción simple de lo que implica el desarrollo de software y también analizar el desarrollo de Java Enterprise.

Back-end Maven


Maven es una herramienta de software para la gestión y construcción de proyectos Java creada por Jason van Zyl, de Sonatype, en 2002. Es similar en funcionalidad a Apache Ant (y en menor medida a PEAR de PHP y CPAN de Perl), pero tiene un modelo de configuración de construcción más simple, basado en un formato XML.

  • Conocimiento : Alto
  • Experiencia 6 años

Back-end Python


Python, al ser un lenguaje de propósito general, se puede usar para diversos propósitos. Es una gran opción para el desarrollo de software, ya que permite a los desarrolladores utilizar grandes frameworks como Django y Flask. Además, se puede utilizar para scripts web, desarrollo de GUI de escritorio o data science.

  • Conocimiento : Alto
  • Experiencia : 8 años

docker


Docker es un sistema operativo para contenedores. De manera similar a cómo una máquina virtual virtualiza (elimina la necesidad de administrar directamente) el hardware del servidor, los contenedores virtualizan el sistema operativo de un servidor.

  • Conocimiento : Alto
  • Experiencia : 8 años
